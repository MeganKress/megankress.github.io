LiDAR Voxel Generator

LiVoxGen is a C++ program that "voxelizes" LiDAR data by partitioning a LAS file into columns of volumetric pixels and counting the LiDAR points inside each voxel. It is hosted on GitHub.

  • Date: Spring 2015
